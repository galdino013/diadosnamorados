# 💖 Surpresa para Giullie Anne

Esse projeto é uma homenagem que eu fiz para a Giullie Anne, minha princesa.

A ideia foi criar uma página simples, bonita e emocionante, reunindo alguns pedacinhos da nossa história em formato de slides. Eu queria fazer algo diferente, que não fosse só uma mensagem comum, mas uma experiência para ela abrir no celular, ver com calma e sentir o carinho em cada detalhe.

---

## ✨ Sobre o projeto

Esse site foi feito como uma surpresa romântica, usando fotos, textos, animações, música de fundo e uma navegação em formato de apresentação.

Cada tela representa um pouco do que eu sinto por ela e algumas lembranças importantes para nós. O objetivo não foi criar algo complexo, e sim algo verdadeiro, leve e especial.

Foi um projeto feito com muito carinho, pensando principalmente em como ela iria ver pelo celular.

---

## 📱 Pensado para celular

Como a ideia principal era ela abrir pelo celular, o layout foi ajustado para funcionar bem em telas menores.

Durante os ajustes, eu me preocupei com:

* boa leitura no celular;
* espaçamento entre os elementos;
* fotos bem posicionadas;
* navegação por toque;
* botões sem atrapalhar o conteúdo;
* visual parecido com uma experiência de stories/slides;
* compatibilidade com iPhone.

---

## 🖼️ Conteúdo da homenagem

A homenagem tem capítulos com momentos e sentimentos importantes, como:

* lembranças de passeios;
* a distância entre São Paulo e Praia Grande;
* momentos que marcaram a nossa história;
* o quanto eu admiro o jeito dela;
* uma declaração final de Dia dos Namorados.

Algumas partes foram escritas de forma mais reservada, porque tem coisas que são muito nossas e não precisam ser explicadas para todo mundo.

---

## 🎵 Música

O projeto também tem música de fundo.

Por limitação dos navegadores, principalmente no celular, a música só começa depois que a pessoa toca na tela ou no botão de música.

O arquivo da música deve estar na raiz do projeto com este nome:

```txt
musica.mp3
```

---

## 🗂️ Estrutura do projeto

A estrutura básica do projeto é essa:

```txt
surpresa-giullie/
│
├── index.html
├── README.md
├── musica.mp3
└── img/
    ├── holambra.jpg
    ├── itanhaem.jpg
    ├── praiagrande.jpg
    └── princesa.jpg
```

As imagens ficam dentro da pasta `img`.

Um ponto importante: na Vercel, o nome dos arquivos precisa bater exatamente com o que está no HTML. Então `holambra.jpg` é diferente de `Holambra.jpg`.

## 🌐 Publicação

O projeto foi publicado pela Vercel, conectado ao GitHub.

Como é um site estático, não precisa de build, backend ou instalação de dependências. É só manter o `index.html`, a pasta `img` e o arquivo `musica.mp3` no repositório.

Configuração recomendada na Vercel:

```txt
Framework Preset: Other
Build Command: vazio
Output Directory: vazio
Install Command: vazio
Root Directory: ./
```
